**DeepSeek API**

**它能做什么：** 通过几行 Python 代码发送请求，让大模型帮你写代码、翻译、解释概念。因为它兼容 OpenAI 的接口，你学一种调用方式就能用很多模型。

**今天可以尝试的一个小操作：** 去 DeepSeek 官网申请一个免费 API Key，然后用 Python 写一个函数，调用 API 让模型解释“PID 控制器的工作原理”，并输出中文。

**来源：** [DeepSeek API Docs](https://api-docs.deepseek.com/)

## 3. 今天的 Codex 使用技巧

**技巧名称：** 用 Codex 写“正则表达式提取器”

**使用场景：** 你在处理自动化日志文件（如西门子 PLC 日志），需要从大量文本中提取“时间”、“错误代码”、“设备 IP”。手工写正则很费时间。

**示例提示词：**  
```
帮我写一条 Python 正则表达式，从下面的日志中提取时间（格式 YYYY-MM-DD HH:MM:SS）、错误代码（如 ERR-1024）、设备 IP：
[2026-07-14 15:32:18] ERR-1024 从设备 192.168.1.100 断开连接
```

**注意事项：** 生成的代码应测试在小样本上，因为 Codex 有时会生成不匹配实际数据格式的表达式。记得在代码里加上 `re.search()` 或 `re.findall()` 调用。

## 4. 今天的英文资料阅读

**英文原句：**  
“Industrial Operations X enables industrial companies to turn a well-automated production into an adaptable and more autonomous production faster and easier.”

**中文解释：**  
“工业运营 X 使工业企业能够将已经高度自动化的生产更快、更容易地转变为可自适应、更自主的生产。”

**关键词：**  
- Industrial Operations X：西门子提出的工业数字化与 AI 交叉的概念  
- adaptable and more autonomous：可自适应、更自主（即不依赖预设，能够根据数据自我调整）

**我应该怎么读懂它：**  
1. 先把长句子拆成短段：“enables... to turn... into... faster and easier.”  
2. 重点记 “autonomous production”（自主生产），这是工业4.0的核心目标，你在《电气自动化》课里学到的传感器、执行器、PLC 都服务于这个目标。  
3. 可以查一下西门子的 Xcelerator 平台，理解“工业运营 X”具体是什么，这对面试很加分。

**来源：** [Siemens Industrial Automation](https://www.siemens.com/global/en/products/automation.html)
